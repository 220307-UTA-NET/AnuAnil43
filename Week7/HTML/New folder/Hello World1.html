Structural Elements

html can support different structure, like tables and lists tool

List

There are a couple of different list elements in html, let's start with an unordered list!

  • the <ul> tag - for unordered lists
  • the <ol> tag - for ordered lists
  • the <dl> tag - for description lists

the ul and ol tags both use the <li> tag to list their items

the dl tag uses the <dt> tag for the term, and the <dd> tag for /p>

Tables

tables are a common way to give an html page structure as well

  • the <table> tag- creates the table
  • the <caption> tag- to list a caption for the table
  • the <tr> tag- to specify the content of a header row
  • the <tr> tag - to specifies the new row
  • the <th> tag - to specify the content of a header cell
  • the <tbody> tag - to specify the content of the body of the table
  • the <td> tag - to specify the content of a cell in the body
  • the <tfoot> tag - to specify the content of the footer row
